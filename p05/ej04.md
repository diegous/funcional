## a) `[[]] ++ xs = xs`
- `[[]] ++ [] = [[]]` // No cumple
- `[[]] ++ [a] = [[],[a]]` // No cumple
**Falso**


## b) `[[]] ++ [xs] = [[],xs]`
- `[[]] ++ [] = [[],[]]` // Si cumple

`xs :: [a]`
**Verdadero**


## c) `[[]] ++ xs = [xs]`
**Falso**, porque `xs` y `[xs]` son de distinto tipo


## d) `[]:xs = xs`
- `[]:[] = [[]]` // No cumple
- `[]:[a] = [[],[a]]` // No cumple
**Falso**


## e) `[[]] ++ [xs] = [xs]`
- `[[]] ++ [[]] = [[],[]]` // No cumple
- `[[]] ++ [[a]] = [[],[a]]` // No cumple
**Falso**


## f) `[[]] ++ xs = []:xs`
- `[[]] ++ [] = [[]] = []:[]` // Si cumple
**Verdadero**


## g) `[xs] ++ [xs] = [xs,xs]`
`xs :: a`  
**Verdadero**


## h) [] ++ xs = []:xs
- `[] ++ [] = []` // No cumple
- `[] ++ [a] = [a] = a:[]` // No cumple
**Falso**


## i) `[[]] ++ xs = [[],xs]`
- `[[]] ++ [] = [[]]` // No cumple
- `[[]] ++ [a] = [[], a]` // No cumple
**Falso**


## j) `[xs] ++ [] = [xs]`
- `[[]] ++ [] = [[]]` // Si cumple
