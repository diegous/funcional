Sea la función `f = foldr (:) []`

* Qué tipo tiene?
```haskell
f :: [a] -> [a]
```

* Reducir la función aplicada a una lista cualquiera.
```haskell
f [1,2,3]
=   def f
foldr (:) [] [1,2,3]
=   def foldr
(:) 1 (foldr (:) [] [2,3])
=   def foldr
(:) 1 ((:) 2 (foldr (:) [] [3]))
=   def foldr
(:) 1 ((:) 2 ((:) 3 (foldr (:) [] [])))
=   def foldr
(:) 1 ((:) 2 ((:) 3 []))
=   reordenamiento de funciones infijas
1 : (2 : (3 : []))
=   eliminación de paréntesis
1:2:3:[]
=   sintaxis de listas
[1,2,3]
```
